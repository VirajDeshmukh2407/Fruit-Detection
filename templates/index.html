About

About This Project

This project develops a deep learning model to detect whether food products — fruits, vegetables, and packaged items — are fresh or expired using image data.

A CNN-based classifier is trained on a custom dataset capturing various stages of decay such as mould growth, colour changes, and texture loss.

The model is integrated into this web application so users can scan and check freshness instantly. This has practical applications in households, supermarkets, and food warehouses to reduce waste and improve food safety.
